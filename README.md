# AI-Based Fitness Activity Recognition

This project is a machine learning application that recognizes human fitness activities
using sensor data and displays predictions through a Streamlit graphical user interface.

## Features
- Upload sensor CSV data
- Predict fitness activity
- Streamlit-based web interface

## Dataset
Human Activity Recognition (HAR) Dataset from Kaggle

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
